## English Description

A high-performance watermark removal tool running entirely in your browser. Deeply optimized for layered watermarks, it provides lightning-fast processing without ever uploading your files to any server.

### 🔗 Live Demo
Visit: [**epiworld.tech**](https://epiworld.tech)

### ✨ Key Features
- **🔒 Privacy First**: All logic runs locally in your browser. Your images never leave your computer—100% private.
- **⚡ High Performance**: Leverages modern browser computing power for multi-tasking and parallel processing.
- **📂 Deep Optimization**: 
  - Specifically optimized for complex layered watermarks (like those found in many AI-generated images).
  - Adjustable **"Watermark Layers"** and **"Removal Intensity"** for precision.
- **📄 Format Support**:
  - Supports batch processing for PNG, JPG, and WebP.
  - Automates watermark cleaning for PDF documents (e.g., from NotebookLM).
- **🎨 Minimalist UI**: Drag-and-drop uploads, real-time comparisons, and instant downloads.

### 🛠 How to Use
1. **Upload**: Drag images or PDF files to the workspace.
2. **Configure**: 
   - **Image Mode**: Adjust layers (1-3) and intensity (0.5-2.0).
   - **NotebookLM Mode**: Optimized for specific PDF cleaning.
3. **Preview**: Instantly check results on the right side of the list.
4. **Save**: Click "Download this file" to save your result.

### 📦 Local Development
```bash
npm install
npm run dev    # Starts dev server on port 3000
npm run build  # Generates production dist folder
```

## 中文介绍

一款基于浏览器端的高性能水印移除工具。针对特定图层式水印进行了深度优化，无需上传服务器，保护隐私的同时，提供极速的处理体验。

### 🔗 在线体验
访问地址：[**epiworld.tech**](https://epiworld.tech)

### ✨ 核心特性
- **🔒 隐私安全**：所有逻辑均在浏览器本地运行，您的素材永远不会离开您的电脑，100% 保护隐私。
- **⚡ 极速并发**：利用现代浏览器计算能力，支持多任务同时并发处理，效率翻倍。
- **📂 深度优化**：
  - 针对复杂图层水印（如 AI 生成图自带的特定标记）进行专项优化。
  - 支持手动调节 **“水印层数”** 与 **“去除强度”**，精准适配不同场景。
- **📄 格式友好**：
  - 支持 PNG, JPG, WebP 图像批量处理。
  - 特别支持针对特定 PDF 文件（如 NotebookLM 生成）的自动化水印清理。
- **🎨 极简交互**：拖拽即上传，列表化实时预览，处理完成即刻保存。

### 🛠 如何使用
1. **上传素材**：将图像或 PDF 文件直接拖入网页，或点击选择。
2. **模式调整**：
   - **图像模式**：根据效果动态调整层数和强度。
   - **NotebookLM 模式**：针对特定 PDF 文档优化。
3. **实时预览**：处理后的效果会实时显示在右侧对比框中。
4. **下载保存**：点击 “下载此文件” 按钮，一键保存成果。
